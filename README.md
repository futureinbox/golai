# golai
GolAI é uma landing page de análise de futebol com foco em estatísticas, odds e sugestões de apostas. Possui design moderno, responsivo e orientado à conversão, servindo como base para um futuro sistema completo com integração de APIs e recursos premium.
# ⚽ GolAI

## 🚀 Funcionalidades
- Interface moderna e responsiva
- Exibição de jogos e estatísticas
- Indicadores de confiança
- Foco em conversão (planos e VIP)

## 🛠️ Tecnologias
- HTML
- CSS

## 📦 Como usar
1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador

Você pode publicar usando:
